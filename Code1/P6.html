Project #6 SleepyQueen

Description

The content of this assignment is to express the sleep time of Jasmine, who loves sleeping most. The emoji represents whether she is awake or asleep. The table represents time.

Design Process

MousePressed and ForLoop are used in this project. Forloop created a visual effect to wrap around the emoji. MousePressed is used to switch Jasmine's state according to the timeline.
