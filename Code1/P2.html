Project #2 Rocket

Description

This is another experimental work, which is to draw a rocket, and I added interaction with the player in the code.

Design Process

This is a more design-conscious work. At the same time, it is also interactive. I made a very simple rocket graphic, and then set the star pattern with MousePressed Function. When the user clicks anywhere on the canvas, stars will appear. This is more like a work done by me and the user.
